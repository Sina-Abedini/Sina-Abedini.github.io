Project Summary

In this project, I Implemented the genetic algorithm using the Python programming language. The problem should be represented so that a 24-bit chromosome can be displayed for each member of the initial population. The initial population members are randomly generated. The fitness function criteria is the most similar to the chromosome 100110101110111110111010. The new generation is created from a combination of parental genes using one of the recombination methods of mutation or inversion.
